
# Stellungnahme-Generator für Finanzierungen

## Anleitung zur Veröffentlichung auf Streamlit Cloud

1. Gehe zu https://streamlit.io/cloud und melde dich mit deinem Account an.
2. Klicke auf "New app" und wähle "From GitHub" oder "From file".
3. Lade den Inhalt dieses ZIP-Ordners hoch oder verlinke dein Repository.
4. Wähle die Datei `chatbot_stellungnahme.py` als Startdatei aus.
5. Klicke auf "Deploy" – die App ist sofort online nutzbar!

## Funktionen
- Auswahl zwischen Wohnbaufinanzierung und Konsumkredit
- Eingabe der Stichworte in ein Formular
- Automatische Generierung einer vollständigen Stellungnahme
